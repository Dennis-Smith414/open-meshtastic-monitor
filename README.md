This is an implementation of a data logging tool for Meshtastic radios. The web interface displays the gps cordations, battery life and gps location of each node on the mesh. This uses a MQTT data broker server and a gateway node to read info from all the other nodes on the mesh. This is currently just starting devlopment and intended as a personal project of mine, however if there is interest I may consider releasing it as a full fleadged package.
